# 🎬 Movie Night Bartender

Telegram бот для выбора фильмов на вечер с умной системой рекомендаций на основе рейтингов IMDb

---

## 🚨 АНТИВИРУСЫ

**Playwright** может вызвать реакцию антивируса. Playwright - это инструмент для эмуляции браузера, необходимый для парсинга IMDb. Он может использоваться как хорошими, так и плохими программами, но сам по себе не является вирусом.

*Выдержка из документации Playwright*

Некоторые антивирусы склонны относить файлы Playwright к классам повышенного риска или хакерским инструментам. Происходит удаление файла и помещение его в карантин.

В случае проблем с антивирусом добавьте папку с проектом в исключения, либо отключите детектирование PUA (потенциально нежелательных приложений).

---


### Установка

1. Клонируйте репозиторий:


> ` git clone https://github.com/bongoket/movie-night-bartender.git ` 

> ` cd movie-night-bartender `  

2. Создайте виртуальное окружение:


> ` python -m venv venv `  

> ` source venv/bin/activate `  # для Linux/Mac

> ` # venv\Scripts\activate  `  # для Windows
  
 
3. Установите зависимости:

> ` pip install -r requirements.txt ` 

> ` playwright install chromium ` 



4. Создайте файл .env и добавьте токен бота:

> `BOT_TOKEN=ваш_токен_от_BotFather  ` 



5. Запустите бота:

> ` python -m bot.main `

## ☑️ РАСПОСТРАНЕННЫЕ ВОПРОСЫ И ПРОБЛЕМЫ:

# После запуска бота ничего не происходит

Убедитесь, что:

- Правильно указан токен в .env

- Не блокирует ли антивирус Python/Playwright

- Запущено виртуальное окружение (venv)

# Бот не находит фильмы / говорит "0 доступных фильмов"

Запустите загрузку фильмов:

> ` python -m services.parser.movie_loader --popular 30 `  

## Не работает Telegram / Discord после установки бота

Бот никак не влияет на работу Telegram или Discord. Проблема в другом.

## Бот перестал работать после обновления

Возможно, изменилась структура IMDb. Попробуйте:

- Обновить Playwright: playwright install chromium

- Переустановить зависимости: pip install -r requirements.txt

## Антивирус ругается на Playwright

Добавьте папку с проектом в исключения антивируса.


## 🗒️ Добавление адресов прочих ресурсов

Список фильмов для загрузки можно расширить, редактируя файл:
> ` services/parser/movie_loader.py  `  - массив ` POPULAR_MOVIES  `
